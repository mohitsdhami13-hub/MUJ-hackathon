# Reality Reimagined — Grief-Tech Ethics Console

> *"Without forgetting it is quite impossible to live at all."* — Friedrich Nietzsche

**A concept built for the MUJ 2026 Hackathon.**

## 🌌 The Concept
In a near-future where people can visit AI-recreated simulations of their deceased loved ones, how do we protect the living from getting lost in the past?

The **Grief-Tech Ethics Console** is a compassionate care dashboard designed for human counselors. It does not exist to police grief or score emotional pain. Instead, it tracks interaction patterns, session cadences, and emotional equilibrium to help a trained human recognize when a grieving person might need a gentle intervention or a tapering plan.

It acts as a mirror for the counselor, not a window into the grieving person's private moments. 

## 🎨 Features & Design
The site was designed with a heavy focus on **emotional resonance, empathy, and high-end cinematic visuals**, moving away from generic "AI UI" tropes.
- **Cinematic 3D Engine**: Powered by a custom Three.js integration featuring an `UnrealBloomPass` for a soft, deeply atmospheric glow that reacts to the cursor.
- **Dynamic Memory Constellations**: An interactive 3D background representing "sessions" as glowing nodes orbiting a central core, mimicking the non-linear path of grief.
- **Interactive Emotion Timeline**: A bespoke 2D HTML Canvas chart tracking emotional state over a 14-week arc, with hover tooltips and pinpointed session notes.
- **Empathetic Micro-interactions**: Soft gradients, glowing hover states, and smooth typography (Cormorant Garamond & Space Mono) designed to feel weightless and respectful.

## 🛠 Tech Stack
This project is built as a highly optimized, dependency-free Single Page Application (SPA):
- **HTML5 & Vanilla CSS**: Utilizing modern CSS Grid/Flexbox, custom properties, and animations. No Tailwind or heavy CSS frameworks were used, ensuring complete control over the visual aesthetic.
- **Vanilla JavaScript**: Handling DOM manipulation, custom charting, and modal logic.
- **Three.js (ES Modules via CDN)**: Used for the immersive WebGL background scene, custom shaders, and post-processing.
